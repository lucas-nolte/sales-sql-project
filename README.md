# Sales Data Analysis Project

This project analyzes a retail sales dataset using SQL and Python to evaluate revenue, product performance, and store-level activity.

## Overview

The dataset contains transaction-level sales data, including product, price, quantity, store location, and date. The analysis focuses on extracting meaningful insights using aggregation, filtering, and sorting techniques.

## Analysis Performed

### Revenue Analysis

- Calculated total revenue accross all transactions
- Computed revenue per transaction using derived columns

### Product Analysis

- Identified top products by total quantity sold
- Identified top products by total revenue

### Store Analysis

- Compared total revenue accross stores
- Analyzed transaction volume per store

### Transaction Insights

- Identified highest-value individual sales

## Key Insights

- A small number of products contribute a significant portion of total revenue
- Store performance varies in both revenue and transaction volume
- High-value transactions can meaninfully impact overall revenue
